# **App Name**: StayOnBeat

## Core Features:

- User Onboarding & Profile Setup: A guided sequence for new users covering legal agreements with interactive toggles, age-gated profile details, and collection of essential health conditions/medications.
- Substance Interaction Risk Assessment Tool: An interactive tool powered by generative AI that evaluates user-selected substance combinations against their health profile to provide personalized risk assessments and detailed interaction explanations.
- Emergency Safety Network Activation: Enables users to generate a shareable QR code linked to their safety profile and activate real-time location sharing with pre-defined awareness teams in critical situations.
- Real-time Event Tracking Map: Displays a high-contrast map interface showing the user's dynamic location, proximity to awareness teams, and integrated quick access to a dose checker during active events.
- Post-Event Secure Data Management: Allows users to selectively purge sensitive GPS and substance intake logs post-event while safeguarding their non-identifiable safety streak progress.
- Staff Awareness & SOS Dashboard: A dedicated and authenticated interface for designated staff to monitor SOS alerts, view crucial user biometrics, and reported substance intake for effective emergency coordination.

## Style Guidelines:

- Background color: Pure Black (#000000) for a deep, dark mode aesthetic, as requested. This establishes a high-contrast base.
- Primary color (for interactive elements): Vibrant Neon Green (#39FF14) to highlight interactivity and convey a techy, alert feel, matching the explicit 'Primary Accents' request.
- Accent color: Bright Yellow-Green (#EBFB3B), an analogous hue to the primary green, to offer visual distinction for secondary highlights or progress indicators while maintaining the energetic palette. User's requested 'bold white' will be used as a dominant text color.
- Headline font: 'Space Grotesk' (sans-serif) for its modern, tech-inspired aesthetic, aligned with the 'bold sans-serif' requirement for headings. Body font: 'Inter' (sans-serif) for high legibility and a neutral, objective feel suitable for detailed legal texts and profile information.
- Clean, line-based white icons for navigation and interactive elements, with key icons like the pulsing heart exhibiting a vibrant neon glow, as requested.
- Structured grid-based layouts with clear left-alignment for textual content and input fields, contrasted with intentional central alignment for critical titles. Interactive elements are arranged for intuitive user flow, featuring vertically stacked buttons and accordion content structures as described.
- Subtle yet impactful animations, including pulsing effects for the heart icon, glowing states for active buttons (like 'GET STARTED'), and neon glow feedback for selected tiles in grids, to enhance user interaction and convey dynamism.
